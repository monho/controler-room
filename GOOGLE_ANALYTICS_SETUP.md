# Google Analytics 설정 가이드

## 1. Google Analytics 계정 생성

1. [Google Analytics](https://analytics.google.com/) 접속
2. "측정 시작" 클릭
3. 계정 이름 입력 (예: MCBL 서버)
4. 속성 이름 입력 (예: MCBL 지휘통제실)
5. 보고 시간대: 대한민국
6. 통화: 대한민국 원 (KRW)

## 2. 데이터 스트림 설정

1. 플랫폼 선택: "웹"
2. 웹사이트 URL 입력: `https://your-domain.com`
3. 스트림 이름: MCBL 웹사이트
4. "스트림 만들기" 클릭

## 3. 측정 ID 복사

1. 데이터 스트림 세부정보에서 "측정 ID" 확인
2. 형식: `G-XXXXXXXXXX`
3. 이 ID를 복사

## 4. 환경 변수 설정

`.env.local` 파일에 측정 ID 추가:

```env
NEXT_PUBLIC_GA_MEASUREMENT_ID=G-XXXXXXXXXX
```

## 5. 추적되는 이벤트

### 자동 추적
- 페이지뷰
- 스크롤
- 아웃바운드 클릭
- 사이트 검색
- 동영상 참여

### 사용자 정의 이벤트
- `streamer_click`: 스트리머 카드 클릭
- `team_filter`: 크루 필터 선택
- `search`: 검색 사용
- `schedule_view`: 일정 조회

## 6. 분석 가능한 데이터

### 사용자 행동
- 방문자 수 (실시간/일별/월별)
- 페이지뷰
- 평균 세션 시간
- 이탈률

### 인기 콘텐츠
- 가장 많이 본 스트리머
- 인기 크루
- 자주 검색되는 키워드

### 사용자 특성
- 지역 (국가/도시)
- 기기 (모바일/데스크톱)
- 브라우저
- 운영체제

### 실시간 데이터
- 현재 활성 사용자
- 실시간 페이지뷰
- 실시간 이벤트

## 7. 대시보드 확인

Google Analytics에서 확인 가능한 주요 리포트:

1. **실시간 리포트**: 현재 사이트 활동
2. **획득 리포트**: 사용자 유입 경로
3. **참여도 리포트**: 페이지 및 이벤트 분석
4. **수익 창출**: 전환 추적 (설정 시)

## 8. 맞춤 대시보드 생성

추천 대시보드 구성:

- 일일 활성 사용자
- 인기 스트리머 TOP 10
- 크루별 조회수
- 검색 키워드 순위
- 시간대별 트래픽

## 9. 개인정보 보호

- IP 익명화 자동 적용
- 쿠키 사용 동의 (필요 시 추가)
- GDPR 준수

## 10. 테스트

개발 환경에서 테스트:
1. 사이트 접속
2. Google Analytics 실시간 리포트 확인
3. 이벤트 발생 확인 (스트리머 클릭, 크루 필터 등)
